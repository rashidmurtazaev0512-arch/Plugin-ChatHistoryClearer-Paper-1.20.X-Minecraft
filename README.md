# ChatHistoryClearer

Автоматически очищает историю чата игроков при выходе с сервера.

**Авторы:** vysbik и doqaax

---

## 📋 Что делает плагин?

Когда игрок выходит с сервера, плагин автоматически очищает его историю чата. Это значит, что при повторном входе игрок не увидит старые сообщения, которые были отправлены во время его предыдущей сессии.

### Как это работает?

1. Игрок выходит с сервера
2. Плагин отправляет ему 100 пустых сообщений
3. Старые сообщения "сдвигаются" вверх и исчезают из видимости
4. При повторном входе игрок видит чистый чат

---

## 🚀 Установка

1. Скачайте `ChatHistoryClearer-1.0.0.jar`
2. Положите в папку `plugins/` вашего сервера
3. Перезапустите сервер

---

## 📝 Команды

### `/clearchat` (или `/cc`, `/chatclear`)

Очищает историю чата.

#### Использование:

```bash
# Очистить чат для ВСЕХ игроков на сервере
/clearchat

# Очистить чат для конкретного игрока
/clearchat <никнейм>
```

#### Примеры:

```bash
/clearchat
/clearchat Notch
/cc
/chatclear
```

## 🔐 Права доступа (Permissions)

| Право | Описание | По умолчанию |
|-------|----------|--------------|
| `chathistoryclearer.use` | Разрешает использовать команду `/clearchat` | OP |
| `chathistoryclearer.bypass` | Защищает от очистки чата (администраторы) | OP |

### Пример выдачи прав (LuckPerms):

```bash
# Выдать право на очистку чата
/lp user Admin permission set chathistoryclearer.use true

# Защитить администратора от очистки
/lp user Admin permission set chathistoryclearer.bypass true
```

## ⚙️ Как работает очистка?

### Механизм:

```java
for (int i = 0; i < 100; i++) {
    player.sendMessage("");
}
```

Плагин отправляет 100 пустых сообщений игроку. Это "сдвигает" старые сообщения вверх за пределы видимости.

### Что очищается?

- ✅ История чата текущей сессии (стрелки вверх ↑)
- ✅ Сообщения, отправленные во время игры
- ❌ История чата между сессиями (хранится на клиенте, сервер не может её контролировать)

## 🎯 Особенности

### 1. Автоматическая очистка при выходе

При выходе игрока с сервера (`PlayerQuitEvent`) его история чата автоматически очищается.

### 2. Глобальная очистка

Администратор может очистить чат для всех игроков командой `/clearchat`.

### 3. Очистка для конкретного игрока

Администратор может очистить чат только для одного игрока:
```bash
/clearchat Notch
```

### 4. Защита администраторов

Игроки с правом `chathistoryclearer.bypass` защищены от очистки чата:
- Их чат не очищается при глобальной очистке
- Их чат нельзя очистить командой `/clearchat <игрок>`

## 📊 Проверка работы

### 1. Проверка автоматической очистки:

1. Войдите на сервер
2. Отправьте несколько сообщений в чат
3. Выйдите с сервера
4. Зайдите снова
5. Нажмите стрелку вверх ↑ — история должна быть пустой

### 2. Проверка команды:

```bash
# Очистить чат для всех
/clearchat

# Очистить чат для конкретного игрока
/clearchat Notch
```

### 3. Проверка прав:

```bash
# Обычный игрок (без прав) не может использовать команду
/clearchat
# ❌ "У вас нет прав для использования этой команды!"

# Администратор (с правом chathistoryclearer.use) может
/clearchat
# ✅ "История чата очищена для всех игроков!"
```

## ⚠️ Важно знать

### Ограничения:

1. **История между сессиями:** Сервер не может контролировать историю чата между сессиями. Она хранится локально на клиенте игрока. Плагин очищает только текущую сессию.

2. **Клиентская сторона:** Если игрок использует моды или плагины на клиенте, которые сохраняют историю чата, плагин не сможет её очистить.

3. **Paper API:** Плагин использует Paper API 1.20. На других ядрах (Spigot, Bukkit) может не работать.

### Рекомендации:

- Используйте вместе с плагином, который отключает сохранение чата на клиенте
- Для полной очистки используйте `/clearchat` + выход игрока
- Выдавайте права только доверенным администраторам
